# Shoe_store

Trang chủ: Danh sách sản phẩm (hiển thị giày theo danh mục hoặc phổ biến).

Giỏ hàng: Danh sách các sản phẩm đã chọn, tính tổng tiền.
Thanh toán: Nhập thông tin khách hàng và phương thức thanh toán.
